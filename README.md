Tyoohar Ghar - Christmas Edition (React + Vite + Tailwind)

This package is a premium Christmas-themed site ready to deploy to Netlify.

Instructions:
1. Unzip and upload all files to a GitHub repo (commit to main).
2. In Netlify, New site -> Import from Git -> connect GitHub and select this repo.
3. Build command: npm run build
   Publish directory: dist
4. If Netlify previously failed, use 'Clear cache and deploy'.

Replace /src/assets placeholder images with real product photos and a high-res christmas-room hero image: 'christmas-room.jpg', 'hero-hampers.png', 'small.jpg', 'medium.jpg', 'large.jpg'.

Tyoohar Ghar - React + Vite + Tailwind starter
--------------------------------------------------

How to run locally:
1. npm install
2. npm run dev
3. Open http://localhost:5173

Build for production:
1. npm run build
2. Deploy the 'dist' folder to Netlify or Vercel.

Notes:
- Replace images in src/assets with real product photos.
- Configure Netlify environment and set site name to your preference.
